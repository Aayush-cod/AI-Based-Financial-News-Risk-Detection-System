## How to Run

1. Download required datasets from Kaggle:
   - Financial PhraseBank
   - Financial News Dataset (2012–2022)

2. Place datasets inside project folder

3. Run notebooks in order:
   - Combining_datasets.ipynb
   - Cleaning_Preprocessing.ipynb
   - Model_Training&Evaluation.ipynb

4. Run Streamlit:
   streamlit run app.py
